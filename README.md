# AG-Portfolio

A premium, high-performance portfolio website built with **React** and **Vite**, featuring smooth animations and a modern design system.

## 🚀 Live Demo

The website is deployed and available at the following locations:

- **Primary (AWS)**: [https://www.ajaygangwar945.xyz](https://www.ajaygangwar945.xyz)
- **Secondary (Vercel)**: [https://ajaygangwar-portfolio.vercel.app/](https://ajaygangwar-portfolio.vercel.app/)

## ✨ Key Features
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop.
- **Modern Stack**: Built with React 19, Vite 8, and Framer Motion for premium animations.
- **High Performance**: Lightning-fast load times and smooth transitions.
- **AWS Infrastructure**: Deployed using AWS Amplify with a custom `.xyz` domain and SSL security.

## 🛠️ Tech Stack
- **Frontend**: [React](https://reactjs.org/), [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: Vanilla CSS with a focus on visual excellence
- **Deployment**: [AWS Amplify](https://aws.amazon.com/amplify/), [Vercel](https://vercel.com/)
